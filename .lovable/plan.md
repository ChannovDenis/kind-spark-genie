
# План реализации: Генератор видео

## Цель
Создать страницу `/studio/generator` — форму для генерации AI-видео с выбором сервиса, темы, голоса, длительности и live-превью результата.

### Компоненты формы

**1. Левая колонка — форма**
- **Сервис** — Select с иконками из `mockData.services`
- **Тема** — Textarea для ввода темы/вопроса
- **Голос AI** — Select с вариантами:
  - Анна (женский, спокойный)
  - Дмитрий (мужской, уверенный)
  - Мария (женский, энергичный)
  - Алексей (мужской, нейтральный)
- **Длительность** — Slider от 1 до 10 минут с отображением значения
- **Стиль подачи** — RadioGroup:
  - Информативный
  - Дружелюбный
  - Профессиональный
- **Кнопка генерации** — с иконкой Sparkles, disabled пока не заполнены обязательные поля

**2. Правая колонка — превью**
- Placeholder-карточка с mock-thumbnail
- Отображение выбранных параметров в реальном времени:
  - Тема (первые 50 символов)
  - Сервис
  - Голос
  - Длительность
- Расчётное время генерации (mock: ~2-5 мин)
- Статус: "Заполните форму" → "Готово к генерации"

**3. Интерактивность**
- Все изменения формы мгновенно отражаются в превью
- Кнопка "Сгенерировать" показывает toast с сообщением об успехе и редиректит на `/studio` через 2 секунды
- Кнопка "Назад к видео" — Link на `/studio`

## Стилистика

- Двухколоночный layout: `grid grid-cols-1 lg:grid-cols-2 gap-8`
- Форма в glass-card с padding
- Превью в отдельной glass-card с акцентным бордером
- Кнопка генерации: `variant="default"` с Sparkles иконкой
- Поля формы: стандартные gap-4 между элементами

---

## Результат

После реализации контент-менеджер сможет:
1. Выбрать сервис для видео (юрист, врач и т.д.)
2. Описать тему видео
3. Выбрать голос AI-диктора
4. Указать желаемую длительность
5. Выбрать стиль подачи материала
6. Видеть live-превью будущего видео
7. Запустить генерацию и получить уведомление
